# Supertonic 3 — Colab Full Working + Fast Speech + Expressions

# ⚡ SPEED CONTROL RULES (IMPORTANT)

To make speech faster:

### ❌ Avoid (slows down speech)

* `<breath>`
* `<sigh>`
* long sentences
* "..." pauses
* paragraph spacing

### ✅ Prefer

* short sentences
* punctuation-based rhythm
* minimal tags:

  * `<laugh>` (safe)
  * `<cough>` (rare)

---

# ⚡ FAST VS SLOW STYLE

### Fast (recommended for assistants)

```text
Hey! That’s crazy. <laugh> Let’s fix it.
```

### Slow (narration/podcast style)

```text
Hey there... I honestly cannot believe this happened... <breath>
```

---

# 🚀 Best practical setup for you

* Voice: `M2` or `M3`
* Tags: only `<laugh>`
* Sentences: short
* No extra line breaks
